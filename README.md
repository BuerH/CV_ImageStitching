## 图像拼接
> 本项目主要内容是使用C++实现简单的视频拼接任务，方便后面学习的参考，除了投影部分，其它代码都是依赖OpenCV已经实现的库函数。

## 项目结构介绍

### CornerDetect
这个文件夹下面实现了一些角点检测任务
已经实现的有以下内容：
- Harris角点检测
- Shi_Tomas角点检测

### FeatureDetect
这个文件夹下面实现了一些特征点检测任务
已经实现的有以下内容：

- SIFT算法
- SURF算法
- ORB算法

### Projection
这个文件夹下面实现了图像投影任务
已经实现的有以下内容：

- 柱面投影
- 球面投影

### SRC

这个文件夹下面放置图像文件，main.cpp主要用来测试已经实现的代码。
